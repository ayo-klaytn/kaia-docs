# 取消

**TxTypeCancel** 取消交易池中具有相同 nonce 的交易的執行。 當已提交的交易看起來有一段時間沒有被處理時，此交易類型就很有用。

<CH.Spotlight>
  <CH.Code>
    ```js TxCancelType.js
    import { createWalletClient, http, privateKeyToAccount, TxType } from "@kaiachain/viem-ext";
    import { kairos } from "viem/chains";
    const senderWallet = createWalletClient({
      chain: kairos,
      transport: http(),
      account: privateKeyToAccount(
        "0x0e4ca6d38096ad99324de0dde108587e5d7c600165ae4cd6c2462c597458c2b8"
      ),
    });
    // Example usage
    (async () => {
      const txRequest = await senderWallet.prepareTransactionRequest({
        account: senderWallet.account,
        type: TxType.Cancel,
      });
      console.log("txRequest", txRequest);
      
      const sentTx = await senderWallet.sendTransaction(txRequest);
      console.log("Cancel tx", sentTx);
    })();

    ```

  ---

  匯入 **@kaiachain/viem-ext** 套件以在 web3 上新增 kaia 功能

  ```js TxCancelType.js focus=1
  ```

  ---

  指定**kairos**連線設定，以連線至 Kaia Kairos 測試網路。

  ```js TxCancelType.js focus=2
  ```

  ---

  使用 **createWalletClient** 設定一個錢包用戶端，並配置 **Kairos 鏈**、**HTTP 傳輸**，以及轉換為帳戶的**寄件者私密金鑰**。

  ```js TxCancelType.js focus=3:9
  ```

  ---

  使用 **prepareTransactionRequest** 建立**取消交易**的交易請求，指定交易類型和寄件者帳戶。
