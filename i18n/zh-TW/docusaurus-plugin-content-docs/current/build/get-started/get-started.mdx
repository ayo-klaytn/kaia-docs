# 開始使用 Kaia

歡迎來到 Kaia！ 無論您是區塊鏈開發新手，還是對 EVM 鏈有豐富經驗，本指南都能幫助您在 Kaia 上快速建構。 我們會一步一步帶您完成設定、工具和您的第一個專案。

遵循這個以目標為基礎的路徑，從零開始部署您的第一個智慧型契約或 dApp。 每個步驟都建立在上一個步驟的基礎上，所以請從頭開始，或跳到您需要的步驟。

<GetStartedCards />

<h2 id="step-1-foundation-setup">步驟 1：地基設定</h2>

探索 Kaia 的 EVM 相容架構、具有 RPC 端點和探索器的主要網路，以及用於高效開發的 Solidity、SDK、Hardhat、Foundry、Thirdweb 和 Remix 等關鍵工具。

[基礎設定](../get-started/foundation-setup.md)

<h2 id="step-2-set-up-wallet--network">步驟 2：設定錢包與網路</h2>

選擇 Kaia Wallet 或 MetaMask 等錢包，新增 Kaia 網路，並建立帳戶。

[如何為 Kaia Networks 設定您的錢包](../wallets/wallet-config/configure-wallet-for-kaia-networks.mdx)

<h2 id="step-3-get-test-kaia">步驟 3：取得測試 KAIA</h2>

從水龍頭為您的錢包注入 testnet KAIA 資金，以支付開發期間的瓦斯費用。

[Get KAIA](../get-started/getting-kaia.md)

<h2 id="step-4-hello-world-with-remix">步驟 4：使用 Remix 的 Hello World</h2>

使用 Remix IDE 部署一個簡單的「Hello World」契約。 我們會使用 KaiaGreeter 樣品作為您的第一份合約。

[KaiaGreeter](../smart-contracts/token-development/samples/kaiagreeter.md)

<h2 id="step-5-create-your-first-token">步驟 5：建立您的第一個代用幣</h2>

使用 Hardhat 建立並部署靈魂標記 (不可轉移的 NFT)。 這包括專案設定、測試和部署。

[使用 Hardhat 部署您的第一個智慧契約](../get-started/hardhat.md)

<h2 id="step-6-build-your-first-dapp-on-kaia">步驟 6：在 Kaia 上建立您的第一個 dApp</h2>

在 Kaia 上使用 Scaffold-ETH 2 建立全堆疊的 dApp。 本教學引導您設定開發環境、為 Kaia 配置 Hardhat、部署和驗證智慧型契約，以及使用 Next.js 建立 React 前端以達到無縫整合。

[使用 Scaffold-ETH 2 建立 dApp](../tutorials/scaffold-eth.md)

<h2 id="step-7-build-a-mini-dapp">步驟 7：建立迷你 dApp</h2>

使用 Unity 或 Cocos 將 Web3 功能整合至 LINE 等平台。 非常適合接觸非加密使用者。

[LINE Mini dApp with Unity](../../minidapps/minidapps.md)
