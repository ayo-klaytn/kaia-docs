# D'cent 生物識別錢包

## 導言<a id="introduction"></a>

D'Cent Wallet 是一款非託管生物識別錢包，旨在提供支持藍牙的硬件和軟件選項，讓您輕鬆管理加密貨幣。

在其[官方網站](https://store.dcentwallet.com/pages/dcent-biometric-crypto-wallet) 獲取 D'CENT 設備
