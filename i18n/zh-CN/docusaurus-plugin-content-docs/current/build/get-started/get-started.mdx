# 开始使用 Kaia

欢迎来到 Kaia！ 无论您是区块链开发的新手，还是在 EVM 链方面经验丰富，本指南都能帮助您快速构建 Kaia。 我们将一步步指导你完成设置、工具和第一个项目。

按照这个基于目标的路径，从零开始部署您的第一个智能合约或 dApp。 每个步骤都建立在上一个步骤的基础上，因此可以从头开始，也可以跳到您需要的步骤。

<GetStartedCards />

<h2 id="step-1-foundation-setup">步骤 1：地基设置</h2>

了解 Kaia 的 EVM 兼容架构、带有 RPC 端点和探索器的主网络，以及用于高效开发的 Solidity、SDK、Hardhat、Foundry、Thirdweb 和 Remix 等关键工具。

[基础设置](../get-started/foundation-setup.md)

<h2 id="step-2-set-up-wallet--network">步骤 2：设置钱包和网络</h2>

选择 Kaia Wallet 或 MetaMask 等钱包，添加 Kaia 网络并创建账户。

[如何为 Kaia Networks 配置钱包](../wallets/wallet-config/configure-wallet-for-kaia-networks.mdx)

<h2 id="step-3-get-test-kaia">步骤 3：获取 KAIA 测试报告</h2>

用水龙头中的 testnet KAIA 为您的钱包充值，以支付开发期间的燃气费。

[Get KAIA](../get-started/getting-kaia.md)

<h2 id="step-4-hello-world-with-remix">步骤 4：使用 Remix 的 Hello World</h2>

使用 Remix IDE 部署一个简单的 "Hello World "合约。 我们将使用 KaiaGreeter 样本作为您的第一份合同。

[KaiaGreeter](../smart-contracts/token-development/samples/kaiagreeter.md)

<h2 id="step-5-create-your-first-token">步骤 5：创建第一个令牌</h2>

使用 Hardhat 创建并部署灵魂令牌（不可转移的 NFT）。 这包括项目设置、测试和部署。

[使用 Hardhat 部署你的第一个智能合约](../get-started/hardhat.md)

<h2 id="step-6-build-your-first-dapp-on-kaia">第 6 步：在 Kaia 上构建您的第一个应用程序</h2>

在 Kaia 上使用 Scaffold-ETH 2 创建全栈 dApp。 本教程将指导您设置开发环境、为 Kaia 配置 Hardhat、部署和验证智能合约，以及使用 Next.js 构建 React 前端以实现无缝集成。

[使用 Scaffold-ETH 2 构建 dApp](../tutorials/scaffold-eth.md)

<h2 id="step-7-build-a-mini-dapp">第 7 步：构建迷你应用程序</h2>

使用 Unity 或 Cocos 将 Web3 功能集成到 LINE 等平台中。 非常适合接触非加密货币用户。

[LINE Mini dApp with Unity](../../minidapps/minidapps.md)

<h2 id="next-steps">下一步工作</h2>

* 有关 Remix 部署的更多信息，请参阅 [Connect Remix to Kaia](../tutorials/connecting-remix.md) 教程。
* 迁移以太坊应用程序？ 请查看 [将以太坊应用程序迁移到 Kaia](../tutorials/migrating-ethereum-app-to-kaia.mdx) 指南，了解快速适配的方法。
* 通过 [Tutorials](../tutorials/tutorials.md) 深入了解实用示例。
* 在 [Secure Wallet Management](../wallets/wallet-config/create-and-manage-wallets-securely.md) 中学习安全实践。
* 查看完整文档，了解[交易](../../build/transactions/transactions.md)、[智能合约](../smart-contracts/smart-contracts.md) 和[节点](../../nodes/nodes.md) 等高级主题。
