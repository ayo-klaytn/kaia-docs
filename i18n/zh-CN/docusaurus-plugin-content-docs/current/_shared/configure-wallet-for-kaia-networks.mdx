# 如何为 Kaia Networks 配置钱包

## 导言

本指南提供了实用的分步说明，指导如何配置领先的软件钱包--Kaia Wallet 和 MetaMask--以连接到 Kaia 主网和 Kairos 测试网。 本指南适用于对加密钱包和网络有基本了解，并需要在 Kaia 上进行构建或测试的开发人员。

您将了解添加 Kaia 网络的最快方法和手动方法，以及如何使用 testnet KAIA 为钱包充值。

## 要求

* 对区块链概念（钱包、网络、交易）有基本了解
* 已安装 Chrome 浏览器或兼容浏览器
* 了解测试网与主网的概念

## 为 Kaia 配置 Kaia 钱包<a id="configure-kaia-wallet-for-kaia" />

[Kaia Wallet](/build/wallets/overview/kaia-wallet.md) 是 Kaia 生态系统的原生浏览器扩展钱包。 虽然它与 MetaMask 共享许多功能，但通过支持独特的交易类型、费用委托交易和网络特有的账户系统，它针对 Kaia 进行了优化，并在网络上提供了无缝的用户体验。

### 在钱包中添加账户

创建 Kaia 钱包时，系统会向您提供一个恢复密语，并自动生成一个账户。 如*创建钱包*部分所示，在启动钱包时，您会看到该账户是您的默认账户。

在某些情况下，您可能需要创建和管理多个账户。 只需在 Kaia 钱包中添加更多账户，就能轻松实现这一过程。

> 请注意，一旦添加账户，就不能永久删除。 它将永远存在于区块链上。

### 安全备份种子词组

您的种子短语是 Web3 钱包的主密钥。 它允许您创建和恢复钱包，包括其中生成的每个账户。 与种子短语绑定的每个账户都有自己的私钥，但种子短语允许访问所有账户--如 *账户 1*、*账户 2* 和 *账户 3*

如果您只有一个账户的私人密钥，则无法访问其他账户。 因此，您的种子短语是需要安全备份的最重要信息。

**该做什么**

* **写下来并安全存储**：手写您的种子短语可防止其在网上被盗。 避免将其存储在任何连接到互联网的设备上。
* **仔细检查您的书写**：确认每个单词的拼写正确，并完全按照提供的顺序排列。
* **离线存储**：物理备份可减少黑客攻击或意外删除的风险。

**避免**

* **不要把它放在显眼的地方**：避免放在明显的地方，如办公桌上的纸条或笔记本电脑上的胶带。
* **不要将其存储在数字位置**：云服务、电子邮件草稿或笔记应用程序容易受到黑客攻击。
* **永远不要与任何人分享**：任何人（包括 Kaia 钱包支持人员）都不得询问您的种子短语。
* **不要改变词语的顺序**：顺序很重要。 任何改动都会使该短语失效。

如果由于某种原因，您忘记备份种子短语或丢失了原始副本，而您的 Kaia 钱包仍未解锁，您仍可以使用密码查看您的*恢复短语*。 请务必立即记录下来并妥善保存。

<Tabs>
  <TabItem value="主网" label="Kaia Mainnet" default>

  <TabItem value="测试网" label="启明星测试网">

    * 在**账户地址**输入框中粘贴您的**钱包地址**。
    * 点击 \*\* 运行水龙头\*\* 按钮，您将收到 50 个 KAIA 测试网。 请注意，每个账户的水龙头每 24 小时可运行一次。

    ![](/img/build/wallets/fund-kaia-faucet.png)
  </TabItem>
</Tabs>

## 将 MetaMask 连接到 Kaia<a id="connect-metamask-to-kaia" />

MetaMask 是 EVM 生态系统中使用最广泛的基于浏览器的钱包之一。 它与 Kaia 地址结构的兼容性使其成为跨多个 EVM 兼容链的开发人员的首选。 本章将告诉你如何为 Kaia 专门配置它。

### 安装 MetaMask

<Tabs>
  <TabItem value="浏览器" label="浏览器扩展" default>
    * 访问 [https://metamask.io](https://metamask.io)
    * 点击菜单栏中的**下载**。
    * 本例中我们将使用 Chrome 浏览器。 (Install
      Chrome)
    * 将 MetaMask
      Extension
      添加到 Chrome 浏览器。
    * 点击 Chrome 浏览器右上角
      的图标，即可启动 MetaMask。
  </TabItem>

  <TabItem value="移动电话" label="移动电话">
    * 我们将从**官方应用程序商店**下载适合您的
      设备/操作系统的 MetaMask。 本例中我们将使用 iOS 系统。
    * 打开**应用商店**，点击右下角的搜索按钮。 在搜索栏中输入 MetaMask。
    * 点击**获取**或点击应用程序名称，查看更多详细信息。 然后，完成常规的 App Store 下载确认流程。
  </TabItem>
</Tabs>

### 创建 MetaMask 钱包

此过程适用于 MetaMask 钱包的浏览器扩展和手机版。

* 点击**创建钱包**按钮。
* 设置密码
* 您将得到一个 12 个字的种子短语；请将其备份到安全的地方。
* 在安全的地方离线备份你的恢复密语！
* 按照所有提示创建您的钱包。

:::note
您只能使用种子短语恢复钱包。 与他人共享您的种子短语可能会导致您失去所有资金。 因此，建议您手动写下来或存储在离线设备中。
:::
