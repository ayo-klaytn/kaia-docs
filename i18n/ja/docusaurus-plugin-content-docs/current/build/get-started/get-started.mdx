# カイアを始めよう

カイアへようこそ！ ブロックチェーン開発の初心者でも、EVMチェーンの経験者でも、このガイドはKaiaを素早く構築するのに役立ちます。 セットアップ、ツール、最初のプロジェクトまで、順を追ってご説明します。

ゼロから最初のスマートコントラクトやdAppをデプロイするまでの、この目標に基づいた道筋に従ってください。 各ステップは最後のステップの上に構築されているので、最初から始めるか、必要なものにジャンプしてください。

<GetStartedCards />

<h2 id="step-1-foundation-setup">ステップ1：ファンデーションのセットアップ</h2>

KaiaのEVM互換アーキテクチャ、RPCエンドポイントやエクスプローラーを備えたメインネットワーク、効率的な開発のためのSolidity、SDK、Hardhat、Foundry、Thirdweb、Remixなどの主要ツールをご覧ください。

[ファンデーションセットアップ](../get-started/foundation-setup.md)

<h2 id="step-2-set-up-wallet--network">ステップ2：ウォレットとネットワークのセットアップ</h2>

カイアウォレットやメタマスクのようなウォレットを選び、カイアネットワークを追加し、アカウントを作成する。

[カイアネットワークス用ウォレットの設定方法](../wallets/wallet-config/configure-wallet-for-kaia-networks.mdx)

<h2 id="step-3-get-test-kaia">ステップ3：KAIAテストを受ける</h2>

テストネットKAIAの蛇口から財布に資金を供給し、開発中のガス料金を賄う。

[KAIAゲット](../get-started/getting-kaia.md)

<h2 id="step-4-hello-world-with-remix">ステップ4：リミックスでハローワールド</h2>

Remix IDEを使ってシンプルな "Hello World "コントラクトをデプロイする。 カイアグリーターのサンプルを最初の契約書として使います。

[カイアグリーター](../smart-contracts/token-development/samples/kaiagreeter.md)

<h2 id="step-5-create-your-first-token">ステップ5：最初のトークンを作成する</h2>

Hardhatを使用してソウルバウンドトークン（譲渡不可能なNFT）を構築し、デプロイする。 これはプロジェクトのセットアップ、テスト、デプロイメントをカバーする。

[Hardhatを使って最初のスマート・コントラクトをデプロイする](../get-started/hardhat.md)

<h2 id="step-6-build-your-first-dapp-on-kaia">ステップ6：Kaiaで最初のdAppを作る</h2>

KaiaでScaffold-ETH 2を使用してフルスタックのdAppを作成します。 このチュートリアルでは、開発環境のセットアップ、Hardhat for Kaiaの設定、スマートコントラクトのデプロイと検証、シームレスな統合のためのNext.jsによるReactフロントエンドの構築について説明します。

[Scaffold-ETH 2を使ったdAppの構築](../tutorials/scaffold-eth.md)

<h2 id="step-7-build-a-mini-dapp">ステップ7：ミニdAppの構築</h2>

UnityやCocosを使って、LINEのようなプラットフォームにWeb3の機能を統合する。 非暗号化ユーザーへのアプローチに最適。

[LINE Mini dApp with Unity](../../minidapps/minidapps.md)
